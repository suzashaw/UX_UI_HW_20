Case Study 1

Bureau of Engraving and Printing Website Redesign

Research

Protopersona

A protopersona was created to understand who the user is for this website.

Usabiity Tests

Usability tests where conducted to see what the users experience was while navigating the site.

Website Heuristics

Website Heuristics and redlining the navigation to the site gave better understanding of user flow.

Problem Statement

The Bureau of Engraving and Printing needs to undergo a website redesign. The website is uninviting, feels stale and antiquated. Information is overwhelming, confusing and difficult to find.

Iterations

Iteration for Desktop HomePage

All iterations are done from the results of user testing. Added U.S. Dept. of Treasury text so the page titles align with the logo text. The BEP logo takes you back to the home page. Added a contact us option to footer. Changed “Social Network” to “Social Media”

Iteration for Mobile App Home Page

The BEP logo on the mobile app now takes you to the home page. A description was added to the icon buttons on the bottom to avoid confusion as to what they where.

Iteration for Mobile App Menu

Contact us was added to the hamburger menu options.
